Assigning (Human) Gene Symbols to a Protein FASTA File...

Welcome! This web tool will use a reciprocal BLAST methodology to assign gene symbols from a human reference protein data set to your protein set of interest.

This online tool has the following, current restrictions:

  • Presently, the pipeline is geared towards vertebrates with a human gene symbol source. Other large divisions coming soon.
  • A maximum of 500K sequences can be submitted.
  • The maximum file size that can be submitted is 500 MB.
  • All sequence identifiers must be unique; otherwise the assignment will fail silently.
  • Presently, default options will be used (E value threshold of 1e-5). This will change in the near future.
